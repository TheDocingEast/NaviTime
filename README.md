**NaviTime** — это легковесный и эстетичный виджет для рабочего стола, предоставляющий информацию о времени и состоянии системы. Проект построен на базе **QML**.

---

## Особенности

* **Минимализм:** Чистый интерфейс, который не отвлекает от работы.
* **Производительность:** Благодаря использованию QML, виджет потребляет минимум системных ресурсов.
* **Информативность:** Отображение текущего времени, даты и ключевых системных метрик.
* **Кастомизация:** Легко правится под свои нужды благодаря декларативному подходу QML.

##  Технологический стек

* **Язык:** QML X Python
* **Фреймворк:** Qt6
